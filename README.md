Spring IOC Project – Dependency Management via Setter Injection (XML)

Overview

This project demonstrates Spring IoC (Inversion of Control) using setter-based dependency injection with XML configuration.

Technologies Used

- Java
- Spring Framework (Core, Context)
- Eclipse IDE
- XML Configuration

Features

- Bean creation using Spring IoC container
- Setter-based dependency injection
- XML-based configuration using "applicationContext.xml"

Project Structure

- "src" → Java source files
- "Cfgs" → Spring configuration files
- "applicationContext.xml" → Bean configuration

How to Run

1. Import the project into Eclipse.
2. Ensure Spring libraries are added.
3. Run the main class.
4. Observe the output in the console.
